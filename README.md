### [NimbleDB](https://github.com/kelvinwambua/nimbledb)
**From-scratch relational database with query optimization and networking**
-  Full RDBMS implementation: storage engine, SQL parser, query optimizer
-  B-tree indexing with O(log n) search and range scans
-  Cost-based query optimizer with join reordering and selectivity estimation
-  Custom binary protocol for client-server communication
-  Slotted page architecture inspired by *Database Internals* by Alex Petrov
-  Concurrent query processing with goroutines and mutex synchronization

**Built for:** Pesapal Junior Dev Challenge 2026  
**Tech:** Go, Custom Binary Protocol, B-trees, SQL Parser  
**Links:** [Repository](https://github.com/kelvinwambua/nimbledb)
